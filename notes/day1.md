# Day 1 — Distributions

Age:      36.7 / 36.0    → flat, symmetric
BMI:      26.6 / 25.98   → RIGHT-skewed
BPSysAve: 118.6 / 116    → RIGHT-skewed (eyes said bell — numbers said no)
Height:   161.9 / 166    → LEFT-skewed... but only because kids are in it
Weight:   70.9 / 72.7    → LEFT-skewed... same reason

ADULTS ONLY (Age >= 20):
Height:   168.79 / 168.7 → SYMMETRIC. Textbook normal.
Weight:   82.2 / 79.5    → RIGHT-skewed. FLIPPED DIRECTION.

KEY LESSONS
1. Skew is named after the TAIL, not the hump.
2. Mean > median = right.  Mean < median = left.
3. Shape is a property of the SAMPLE, not the variable.
   Weird shape? Ask WHO is in the data before you ask about the maths.
4. My eyes said BPSysAve was normal. The numbers said skewed.
   Numbers win.
